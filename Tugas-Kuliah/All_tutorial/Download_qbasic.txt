
1. Download DOSBOx di https://www.dosbox.com( namun jangan di download jika ingin download qbasic langsung dari qb64.net atau Microsoft Store). Pastikan download sesuai OS (windows atau fedora misalnya), lalu install.

2. Download qbasic di 

a. untuk windows 32/64 bit dengan DOSBox : 
	https://www.qbasic.net/en/qbasic-downloads/DOS/Windows-Solutions.htm
	pilih Basic 2 DOSBox (untuk windows, untuk os lain bisa download langsung compilernya)

b. untuk windows 64 bit tanpa DOSBox:
	Download di https://www.qb64.org, download sesuai OS, kemudian install

c. jika anda user windows 10, bisa langsung download qbasic dari Microsoft Store (free), kemudian install


3. jika anda mendownload dengan cara b atau c, seharusnya qbasic sudah dapat dipakai setelah instalasi. jika anda mendownload dengan cara a, maka buka file BASIC2DOSbox.zip yang tadi anda download, klik BASIC2DOSBox, lalu eksekusi basic2dos.exe.

isi Path to DOSBox executable dengan direktori dari DOSBox, misalnya 
C:\Program Files\DOSBox-0.74-2\DOSBox.exe

isi Path to basic destination folder, isi sesuai direktori yang diinginkan, untuk versinya bisa di install sesuai keinginan.

Selesai !!! :)
